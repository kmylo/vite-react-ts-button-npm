## Get your figma API key

https://www.figma.com/developers/api

{/* ![Get access token](https://klaufel.com/tokens/img/get-access-token.png) */}

You can clone our Git repository:
`$ git clone git@github.com:kmylo/vite-react-ts-button-npm.git`

Install dependencies
`$ npm install`

And then, launch storybook
`$ npm run storybook`

Modify config with your figma API key and figma id
`$ input-design-tokens.json`

{/* ![Create config](https://klaufel.com/tokens/img/create-config.png) */}

Generate design tokens as variables

`$ npm run tokens`

😊 Enjoy!
